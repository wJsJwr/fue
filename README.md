# fue
## frequently-used expression

**一个自由开放的常用表达分享平台**

## Dependency
```
gem 'sinatra', '~> 2.0'
gem 'sqlite3', '~> 1.3'
gem 'sanitize', '~> 4.5'
gem 'thin', '~> 1.7', '>= 1.7.2'
gem 'uglifier', '~> 3.2'
gem 'rake', '~> 12.1'
```

## First run
1. `bundle install`
2. `rake install`
3. `ruby fue.rb` and enjoy
